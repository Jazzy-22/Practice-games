# **Galaxy_Rider** 

---

<br>

## **Description 📃**
- Ride sick lines though the galactic bowl system adding to your star collection along the way. 
- Try to finish each course as fast as you can!

## **How to play? 🕹️**
- Controls:
	- Click to start
	- `Left`/`Right` -> Move
	- `Up` -> Ollie
	- `Down` -> Pump (Become heavy)
	- `Enter` -> Select Level
	- `R` -> Restart
	- `Esc` -> Menu
